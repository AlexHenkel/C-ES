Operadores

Muchas veces necesitamos realizar cálculos matemáticas en nuestros programas. Debido a ello, se proporcionan las siguientes operaciones matemáticas que puedes realizar sólo entre números de tipo numero y decimal:

  1. +: para sumar dos números
  2. -: para restar un número de otro
  3. *: para multiplicar dos números
  4. /: para dividir un número de otro

Ilustremos cómo funcionan los operadores con el siguiente programa:

      
   programa

   variables:
   numero numero1, numero2;
   decimal decimal1;

   {
      numero1 = 4
      numero2 = 2
      decimal1 = 2.5

      imprimir (numero1 + numero2)
      imprimir (numero1 - numero2)
      imprimir (numero1 * numero2)
      imprimir (numero1 / numero2)

      imprimir (numero1 + decimal1)
      imprimir (decimal1 - numero2)
   }

El código anterior mostraría en pantalla:

6
2
8
2
6.5
0.5

Operador + con texto

El operador de suma puede ser utilizado con operandos de tipo texto. En este caso, el resultado es la concatenación de los operandos:

      
   programa

   variables:
   texto hola, mundo;

   {
      hola = "¡Hola"
      mundo = ", mundo!"
      imprimir (hola + mundo)
   }

El código anterior mostraría en pantalla:

¡Hola, mundo!
