< Bienvenido a C-ES

Inicio de programa

La tradición sugiere que el primer programa en un nuevo lenguaje de programación debe de imprimir las palabras "¡Hola, mundo!" en la pantalla. Con C-ES esto se puede hacer de la siguiente forma:

         
      programa

      {
         imprimir("¡Hola, mundo!")
      }

Ahora vamos a entender cada pieza del programa anterior.

Programa

Todo programa escrito con C-ES empieza con la palabra reservada programa.

Bloque principal

Otra elemento que también debe de estar presente es el "bloque principal", que consiste de los caracteres { y }.


Es válido tener el bloque principal vacío, pero debe de estar presente. Entonces, cualquier programa debe de tener mínimo lo siguiente:

         
      programa

      {
      }

Comentarios

A veces es necesario dejar anotaciones en el código, ya sea para explicar funcionalidad de tu programa o proporcionar información extra. Para ello, puedes usar comentarios. Simplemente tienes que empezar tu comentario con los caracteres //.


Por ejemplo:

         
      // Esto es un comentario.
      programa

      {
         // Esto también.
         imprimir("¡Hola, mundo!")
      }

Debido a que los comentarios son ignorados al procesar tu código, puedes ponerlos en cualquier lugar.
