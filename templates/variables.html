Variables

Una variable asocia un nombre a un valor de cierto tipo.


Los tipos de datos disponibles son:

  1. numero: para guardar números enteros como 4
  2. decimal: para guardar números decimales como 2.5
  3. texto: para guardar palabras como "Hola"
  4. binario: para guardar ya sea verdadero o falso

Para declarar variables debes de hacerlo después de la palabra programa, especificando primero el tipo, el nombre de la variable y terminando con un punto y coma: 

      
   programa

   variables:
   numero miPrimerVariableDeNumero;
   texto miPrimerVariableDeTexto;

   {
   }

Puedes declarar múltiples variables de un mismo tipo en una sola línea, separadas por comas:

      
   texto miPrimerVariableDeTexto, otraVariableDeTexto;

Una vez declaradas las variables, puedes asignarles un valor dentro del bloque principal:

      
   {
      miPrimerVariableDeNumero = 5
      miPrimerVariableDeTexto = "Hola"
   }

Listas

A veces es necesario agrupar un conjunto de variables, para estas situaciones puedes usar las listas.


Una lista es una colección secuencial de elementos del mismo tipo, de tamaño fijo. Puedes declarar una lista de la siguiente forma: 

      
   // Lista que contiene 5 números.
   lista de numero de 5 miListaDeNumeros;

Puedes declarar múltiples listas de la misma forma que declaras múltiples variables.
