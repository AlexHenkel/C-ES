Variables

Una variable asocia un nombre a un valor de cierto tipo.


Los tipos de datos disponibles son:

  1. numero: para guardar números enteros como 4
  2. decimal: para guardar números decimales como 2.5
  3. texto: para guardar palabras como "Hola"
  4. binario: para guardar ya sea verdadero o falso

Para declarar variables debes de hacerlo después de la palabra programa, especificando primero el tipo, el nombre de la variable y terminando con un punto y coma: 

      
   programa

   variables:
   numero miPrimerVariableDeNumero;
   texto miPrimerVariableDeTexto;

   {
   }

Puedes declarar múltiples variables de un mismo tipo en una sola línea, separadas por comas:

      
   texto miPrimerVariableDeTexto, otraVariableDeTexto;

Una vez declaradas las variables, puedes asignarles un valor dentro del bloque principal:

      
   {
      miPrimerVariableDeNumero = 5
      miPrimerVariableDeTexto = "Hola"
   }

Ahora que las variables ya tienen un valor, puedes hacer referencia a ellas para usarlas en operaciones o expresiones, de las cuales hablaremos más adelante. Para usarlas, simplemente tienes que poner el nombre de la variable, por ejemplo, para mostrar el valor de miPrimerVariableDeNumero en la pantalla lo haríamos de la siguiente forma:

      
   {
      imprimir(miPrimerVariableDeNumero)
   }

El código anterior mostraría en pantalla:

5

Listas

A veces es necesario agrupar un conjunto de variables, para estas situaciones puedes usar las listas.


Una lista es una colección secuencial de elementos del mismo tipo, de tamaño fijo. Puedes declarar una lista de la siguiente forma: 

      
   // Lista que contiene 5 números.
   lista de numero de 5 miListaDeNumeros;

Puedes declarar múltiples listas de la misma forma que declaras múltiples variables.


Una vez que has declarado una lista, puedes asignarle valores indicando el elemento al que le quieres asignar un valor. Sin embargo, en el contexto de las listas, el primer elemento tiene la posición 0, el segundo la posición 1, y así consecutivamente.


Por último, tienes que hacer uso de la función agregar a (en la siguiente lección hablaremos más sobre funciones) de la siguiente forma:

      
   {
      agregar a (miListaDeNumeros, 5, 0)
      agregar a (miListaDeNumeros, 10, 1)
      agregar a (miListaDeNumeros, 15, 2)
      agregar a (miListaDeNumeros, 20, 3)
      agregar a (miListaDeNumeros, 25, 4)
   }

Después de ejecutar el código anterior, el primer elemento de la variable miListaDeNumeros tendría el valor de 5, el segundo elemento tendría el valor de 10, el tercer elemento tendría el valor de 15, el cuarto elemento tendría el valor de 20, y el último elemento tendría el valor de 25.


De la misma forma que puedes usar el valor de las variables, puedes usar el valor de un elemento de la lista. Para ello, tienes que hacer uso de la función acceder e indicar la posición del elemento que quieres, de la siguiente forma:

      
   {
      imprimir (acceder (miListaDeNumeros, 1))
   }

El código anterior mostraría en pantalla:

10
