Control de flujo

Por omisión, el código que compone tu programa se ejecuta de manera secuencial, de arriba hacia abajo. Sin embargo, en ocasiones es necesario omitir ciertas partes de tu programa si existe cierta condición, o incluso repetir una parte de tu programa si otra condición existe.


Para estas situaciones, haremos uso del enunciado si sucede y mientras sucede. Dichos enunciados harán necesario el uso de operadores que nos permiten comparar dos operandos.

Los operadores de comparación disponibles son:

  1. >: compara si la expresión de la izquierda es mayor a la expresión de la derecha
  2. >=: compara si la expresión de la izquierda es mayor o igual a la expresión de la derecha
  3. <: compara si la expresión de la izquierda es menor a la expresión de la derecha
  4. <=: compara si la expresión de la izquierda es menor o igual a la expresión de la derecha
  5. ==: compara si la expresión de la izquierda es igual a la expresión de la derecha
  6. !=: compara si la expresión de la izquierda es diferente a la expresión de la derecha

Enunciado si sucede

Este enunciado permite evaluar una expresión, y en base al resultado, ejecutar cierto código. Por ejemplo, si quisiéramos imprimir un mensaje de "aprobado" si el resultado de un examen fue mayor a 7, y un mensaje de "reprobado" en caso contrario, lo haríamos así:

      
   programa

   variables:
   numero resultado;

   {
      resultado = 8

      si sucede (resultado > 7) realiza {
         imprimir ("aprobado")
      } no sucede {
         imprimir ("reprobado")
      }
   }

aprobado

Podemos evaluar más de una expresión de la siguiente forma:

      
   programa

   variables:
   numero resultado;

   {
      resultado = 7

      si sucede (resultado > 7) realiza {
         imprimir ("aprobado")
      } o si sucede (resultado == 7) realiza {
         imprimir ("aprobado por poco")
      } no sucede {
         imprimir ("reprobado")
      }
   }

aprobado por poco

Enunciado mientras sucede

Podemos ejecutar un pedazo de código hasta que cierta condición se cumpla de la siguiente forma:

      
   programa

   variables:
   numero repeticionActual, repeticionesMaximas;

   {
      repeticionActual = 1
      repeticionesMaximas = 5

      mientras sucede (repeticionActual <= repeticionesMaximas) realiza {
         imprimir (repeticionActual)
         repeticionActual = repeticionActual + 1
      }
   }
