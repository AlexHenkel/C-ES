Funciones

Mientras desarrollamos programas más complejos nos encontramos con la necesidad de ejecutar el mismo código varias ocaciones. En lugar de repetir las mismas líneas de código en todo el programa, podemos encapsularlas en una función, e invocar esa función mediante su nombre las veces que necesitemos ejecutar el código que contiene.


C-ES proporciona al programador la habilidad de desarrollar sus propias funciones, pero también se incluyen funciones ya implementadas cuyo funcionalidad resulta de mucha utilidad. En esta lección explicaremos ambos tipos de funciones.


Funciones propias

Por cada función tienes que especificar la palabra funcion, seguido por el nombre de la función, un conjunto de paréntesis y un conjunto de llaves. Dentro del conjunto de llaves puedes declarar variables y usar otras funciones: 

      
   funcion imprimeUnNumero () {
      numero numero1;
      numero1 = 5
      
      imprimir (numero1)
   }

Para poder ejecutar el código de tu función, tienes que "llamarla", y lo haces poniendo el nombre de la función a ejecutar seguido por paréntesis: 

      
      imprimeUnNumero()

Para declarar tus propias funciones debes de hacerlo después de la palabra programa, y en caso de haber declarado variables después de la declaración de variables. Un programa completo usando la función anterior sería:


      
   programa

   funciones:
   funcion imprimeUnNumero () {
      numero numero1;
      numero1 = 100
      
      imprimir (numero1)
   }

   {
      imprimeUnNumero()
   }

El código anterior mostraría en pantalla:

100

Funciones del lenguaje

Como se mencionó, se proporcionan funciones que son de utilidad y usadas de manera recurrente en programación.

Las funciones disponibles son:

  1. leer (nombreVariable): guarda en la variable el valor leído del teclado
  2. imprimir (nombreVariableOValor): muestra en pantalla el valor de la variable especificada
  3. agregar a (lista, valor, posición): guarda un valor en el elemento especificado de la lista
  4. acceder (lista, posición): obtiene el valor del elemento especificado de la lista
  5. quitar ultimo de lista (): quita el valor actual que contenía el último elemento de la lista
  6. numero aleatorio (de variableONumeroIndicandoLimiteInferior, variableONumeroIndicandoLimiteSuperior): regresa un numero aleatorio cuyo valor está entre entre los límites establecidos
